# Talisman Dictionary

Simple static web application to serve as a mobile dictionary for Talisman Board Game while playing with French language version.

## How to use it

1. Go to https://talisman-dict.github.io/
2. Search for a card (either by English or French card name), e.g. search for *Élémentaire*, or *Elementaire*
3. You can use suggestions to find cards faster.
4. Get to know your card details.

The card details and images were scraped from old Talisman wiki page http://www.talismanwiki.com/.


## Why would you even want play French version of a board game?

As of 2026 our fellow French people are selling both board game and it's expansions the cheapest (at least in Europe). For example in Poland, one would have to pay 200-300 EUR for a single Talisman Extension, but on French Amazon you could find the same in French for 50 EUR. Money talks!
